ИСПРАВЛЕНИЯ СТАТУСОВ И ГАЛОЧЕК
================================

Что было исправлено:

1. СТАТУСЫ ПОЛЬЗОВАТЕЛЕЙ (ЗЕЛЕНЫЙ КРУЖОК)
   - Теперь статусы сохраняются в базу данных
   - При загрузке списка контактов показываются актуальные статусы
   - Цвета стали ярче и заметнее:
     * Онлайн: яркий зеленый (#22C55E)
     * Не беспокоить: яркий красный (#EF4444)
     * Офлайн: светло-серый (#9CA3AF)
   - Статусы обновляются в реальном времени для всех пользователей

2. ГАЛОЧКИ СООБЩЕНИЙ (✓ и ✓✓)
   - Статусы сообщений теперь сохраняются в базу данных
   - При перезагрузке приложения галочки остаются
   - Обновления статусов работают в реальном времени:
     * ✓ (серый) = Отправлено
     * ✓✓ (серый) = Доставлено
     * ✓✓ (синий) = Прочитано
   - НОВОЕ: При открытии диалога все непрочитанные сообщения автоматически 
     помечаются как прочитанные, и галочки обновляются у отправителя

3. ОБНОВЛЕНИЯ В РЕАЛЬНОМ ВРЕМЕНИ
   - Когда пользователь выходит, его статус меняется на "Офлайн"
   - Все подключенные клиенты видят изменение статуса сразу
   - Сообщения обновляют статус автоматически
   - При открытии диалога с историей, все входящие сообщения помечаются 
     как прочитанные

КАК ПРОВЕРИТЬ:

1. Запустите сервер: dotnet run --project ChatApp.Server
2. Запустите два клиента на разных компьютерах
3. Войдите под разными аккаунтами
4. Проверьте:
   - Зеленый кружок у онлайн пользователей
   - Отправьте сообщение - должна появиться одна галочка (✓)
   - Получатель должен увидеть сообщение - у отправителя появятся две галочки (✓✓)
   - Если получатель откроет диалог, галочки станут синими (прочитано)
   - Выйдите из одного клиента - кружок должен стать серым
   - НОВОЕ: Отправьте сообщения когда получатель офлайн, затем получатель 
     заходит и открывает диалог - галочки должны стать синими
